# 汉 HanGo — Aprende chino sin complicarte

Plataforma de chino mandarín para hispanohablantes con IA.

## Features
- 🎓 **Tutor IA** — Lecciones visuales interactivas
- 💬 **小美** — Conversación por voz con amiga china virtual
- 📖 **Historias** — Cuentos generados por IA adaptados a tu nivel
- ✍️ **Trazos** — Practica escritura de caracteres
- 😂 **Memes** — Aprende cultura china con humor

## Setup local

```bash
npm install
npm run dev
```

## Deploy en Vercel

1. Sube este repo a GitHub
2. Conecta a Vercel (vercel.com)
3. Agrega variable de entorno: `ANTHROPIC_API_KEY`
4. Deploy automático
